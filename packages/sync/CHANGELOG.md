# Changelog

All notable changes to `@wangai/sync` will be documented in this file.

## 0.1.0

- Initial release
- Unified baseline model (`skills + agents`)
- Commands: `login`, `init`, `push`, `pull`, `status`
- Markdown-only baseline extraction (`*.md`)
- Platform enable/disable support during `init`
- Jianguoyun WebDAV integration
- Bilingual CLI output (`--lang zh|en`, default `zh`)
